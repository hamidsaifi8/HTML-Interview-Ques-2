Day2 Interview Question Q- Difference between HTML tags and elements Q-Explain br tags Q-Explain how many different types of lists we have in html Q-What are semantic tags

Q-1. Difference between HTML tags and elements.

HTML-Tags:- Are the keywords on web page that defines the structure of page. exp - HTML tag,body tag, head tag etc. They always have open and closing tag.
HTML Elements: - Are the keywords with open and closing tags that provide the addional information in HTML webpage. exp - h tag, p tag, a tag etc.

Q-2. Explain br tags

br tags:- its a self closing tag and mainly use to break the lines and adding the space also in between lines

Q-3. Explain how many different types of lists we have in HTML.

In HTML mainly we do have 3 type of list.

a. Unordered List
- As per name suggested, it dont' have any order and denotes as (Disc, Circle, Square & None)

b. Ordered List
- As per name suggested, its in the ordered form and denotes as (Number, A,a, I,i)

c. Description List
- Its basically used for description of anything. it contain the heading and description of item.

Q-4. What are semantic tags?

Semantic Tags:- are used to provide the structure/meaning of web page to the browser and developer, so the they can understand the structure of html page.. exp - Header, Navigation, Section, Footer etc.

Q-5. Write code to create an ordered list with roman numbers.

Ordered list in upper roman
<ol type="I"> <li>Tea</li> <li>Coffee</li> <li>Milk</li> </ol>
Ordered list in lower roman
<ol type="i"> <li>Tea</li> <li>Coffee</li> <li>Milk</li> </ol>

Q-6. Write code to create an unordered list with circle.

<ul type="Circle"> <li>Tea</li> <li>Coffee</li> <li>Milk</li> </ul>
